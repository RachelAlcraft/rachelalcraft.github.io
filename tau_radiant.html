Tau and density (private link)

Description

Data for the investigation of tau geometry and the electron density

Best Supported Dataset

A dataset is obtained from ultrahigh-resolution xray crystallographic structures that will have sufficient experimental evidence to inspect and validate all tau variation against the electron density.

The set is first created from pdbs <1A non-homologous at 90% with >40 residues.

From this only residues with no multiple occupants are accepted. A further restriction is put on the bfactor of a residue to be no more than 1.3x the average bfactor of the structure.

There are 3,630 observations of glycine tau angles in this good data set. From these a further restriction on quality is added - every atom is analysed to verify it is at a density peak in the structure. Where this is not true, the residue is rejected. 760 GLY observations pass this electron density test and make it into the "best supported" set.

The geoemetry is compared across "all" and "best supported" data sets.

Note - A category is invented that attempts to split the values up into 5 logical groups.

  • Geometry comparison: "All" vs "best supported" - This shows histgrams and scatter plots for tau, N:N+1, psi and phi

Taking a selection of residues at the most extreme tau ends in the best supported categories, the electron density is visualised for supporting evidence of the accuracy of the pdb tau calculation. The possible causes of variation on the tau angle can be inspected by the seperation into these categories.

At the end of each set of visualised electron density data, an average image is created - this gives some idea of the differences between the categories.

Looking for tau improvements in density peaks

This is a set of data created prior to the "best supported" data which shows the need to look at better supported data.

The 5 sets of data below are an attempt to recalculate tau when the density peak is not at the purported atom centre

5 sets of pdbs in different categories

  • Set 1: Extreme tau values around PSI=0
  • Set 2: Long N:N+1 with +ve PSI near 180
  • Set 3: Long N:N+1 with -ve PSI near 180
  • Set 4: Short N:N+1 with PSI near <0
  • Set 5: Short N:N+1 with PSI near >10
  • Set 6: Highest resolution with tau differences
