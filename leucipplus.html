LeucipPlus : Atomic Density Explorer

Summary

The Leucippus suite consists of the python library LeucipPy and a c++ utility LeucipPlus.
LeucipPlus is primarily concerned with re-refinement, structure factors and electron density.
A preliminary version of the integrated environment can be found online as part of my PhD project.
The code is entirely under developement and not released.
